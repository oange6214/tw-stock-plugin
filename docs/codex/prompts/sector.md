# Sector Prompt

Use this prompt when the user wants a sector overview or industry research note.

## Execution Steps

1. Read `equity-research/commands/sector.md`.
2. Read `equity-research/skills/sector-overview/SKILL.md`.
3. Use `tw-stock-mcp` for market context and company support data.
4. Describe the industry chain, demand drivers, supply dynamics, key names, and cycle view.
5. Save the output to the workspace `reports/{sector}-overview.md` unless the user requests another path.
