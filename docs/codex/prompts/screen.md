# Screen Prompt

Use this prompt when the user wants to screen Taiwan stocks by strategy or factor conditions.

## Execution Steps

1. Read `equity-research/commands/screen.md` and, if relevant, `taiwan-trading/commands/select.md`.
2. Read the closest related skill for screening or idea generation.
3. Use `tw-stock-mcp` to gather market and stock-level data.
4. Explain the screen logic before presenting results.
5. Save the output to the workspace `reports/screen-{theme}-{date}.md` unless the user requests another path.
