# Initiation Prompt

Use this prompt when the user wants an initiating coverage note or first-pass company report.

## Execution Steps

1. Read `equity-research/commands/initiate.md`.
2. Read `equity-research/skills/initiating-coverage/SKILL.md`.
3. Use `tw-stock-mcp` to gather market context and stock data.
4. Build a structured initiation note covering business quality, growth drivers, risks, catalysts, and valuation context.
5. Save the output to the workspace `reports/{ticker}-initiation.md` unless the user requests another path.
