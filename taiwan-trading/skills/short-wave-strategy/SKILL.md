# 台股兩週短波段交易系統

## 一、選股大方向

兩週的波段操作必須搭上市場的順風車，選擇資金正在匯聚的主流族群。

### 族群選擇優先順序
1. **大方向鎖定強勢產業**
   - 使用 tw-stock-agent 查詢各族群近 5 日漲幅排行
   - 鎖定資金持續流入的主流族群（AI、半導體、伺服器、綠能等）
   - 避開族群輪動末段的標的

2. **個股條件（基本過濾）**
   - 股價 > 10 元（避免低價股操控風險）
   - 市值 > 50 億（流動性充足）
   - 近 3 個月平均日成交額 > 1 億（可進出）
   - 財報無重大虧損（近兩季 EPS > 0）

## 二、技術面選股條件

### 必要條件（全部符合才進場）

**量價突破型**
- 股價突破近 20 日高點，當日成交量 > 5 日均量 1.5 倍以上
- 站上 5MA、10MA、20MA（均線多頭排列）
- RSI(14) 在 50-75 之間（強勢但未過熱）

**型態確認**
- 優先選擇：底部打底後突破、整理後爆量突破、強勢股回測均線支撐
- 避開：高檔爆量長黑、連續大漲後追高、K 線型態頭部形成

### 加分條件（2 項以上加分）
- MACD 黃金交叉或翻正
- KD 值低檔鈍化後向上
- 布林通道突破上軌
- 月線維持多頭排列

## 三、籌碼面確認

使用 tw-stock-agent 查詢籌碼數據：

**外資動向**
- 外資連續 3 日以上買超（優先選擇）
- 外資持股比例近期上升

**投信動向**
- 投信連續買超（配合外資更佳）
- 投信買超金額 > 1000 萬/日

**融資動向**
- 融資維持穩定或小幅增加（融資爆增要小心）
- 融資增加但股價也上漲：OK
- 融資增加但股價下跌：危險信號

**大戶持股**
- 大戶（400 張以上）持股比例上升

## 四、進場策略

### 進場時機
- **最佳**：突破當日或次日收盤前確認量能
- **次佳**：突破後回測均線支撐、成交量縮小時
- **避免**：追高（距突破點漲超過 5%）

### 部位管理
- 單筆部位：總資金的 15-25%
- 最多同時持有 4 支
- 總曝險：不超過總資金的 80%

### 進場範例

**案例：台積電（2330）突破進場**

| 項目 | 內容 |
|------|------|
| 進場日期 | D 日（突破當日收盤前） |
| 進場價 | 855 元（當日收盤確認量能） |
| 停損價 | 795 元（-7%，跌破 20MA） |
| 第一停利 | 940 元（+10%，出清 1/3） |
| 第二停利 | 980 元（+15%，出清 1/3） |
| 時間停利 | D+8 個交易日若未達目標，出清剩餘 1/3 |
| 部位大小 | 總資金的 20%（4 支持倉中的一支） |

**進場確認清單：**
- [x] 股價突破近 20 日高點（850 元）
- [x] 當日量 > 5 日均量 × 1.5（量比 1.8x）
- [x] 5MA / 10MA / 20MA 多頭排列
- [x] RSI(14) = 62（在 50-75 區間內）
- [x] MACD 翻正
- [x] 外資連續 4 日買超
- [x] 市值 > 50 億，日均成交額 > 1 億

## 五、出場策略

### 停損條件（任一觸發即出場）
- 跌破停損價（進場價 -7% 以內）
- 跌破 10MA 且成交量放大
- 籌碼出現大幅賣超（外資連續 2 日大賣）
- 距離進場超過 10 個交易日

### 停利條件
- **時間停利**：持有 8-10 個交易日後，若無明顯方向，出清 50%
- **獲利停利**：達目標漲幅（10-15%）分批出場
- **技術面轉弱**：跌破 5MA 且量縮

### 出場執行原則
- 停損：立即執行，無需等待
- 停利：分 2-3 批出場（1/3 在第一目標、1/3 在第二目標、1/3 用時間停利）

## 六、風控規則（不可違反）

1. **絕對不加碼虧損部位**
2. **停損不移動**（設定後不往下移）
3. **大盤跌 2% 以上當日**：降低新倉部位或暫停進場
4. **持有超過 3 支虧損部位**：停止新增部位，先處理虧損
5. **單週虧損超過總資金 5%**：停止操作，檢討策略

## 七、使用 tw-stock-agent 的工作流程

### 每日例行工作

**盤前（09:00 前）**
1. 用 `get_market_overview` 查詢大盤環境與外資期貨淨部位
2. 確認加權指數在均線相對位置（多頭 / 盤整 / 空頭）
3. 掃描持倉個股最新籌碼變化（外資、投信昨日動向）
4. 確認今日是否有重要財報發布或法說會

**盤中監控**
1. 監控各持倉是否觸及停損條件（設手機提醒）
2. 留意個股成交量異常（爆量下殺 = 立即評估）
3. 大盤跌 2% 以上：暫停新倉，評估現有部位

**盤後（15:30 後）**
1. 更新各持倉損益與持有天數
2. 執行當日選股掃描（大盤環境允許時）
3. 確認明日潛在進場候選名單
4. 記錄操作日誌（進出場理由、情緒狀態）

### 選股掃描流程

**Step 1：大盤環境確認**
```
get_market_overview → 確認加權指數站上 5/10/20MA
確認外資期貨淨多單 > 0
若環境偏弱（指數跌破 20MA），停止掃描、暫停進場
```

**Step 2：族群動能排行**
```
查詢各族群近 5 日漲幅排行
選出前 3 強勢族群
確認族群資金持續流入（非輪動末段）
```

**Step 3：個股技術篩選**
```
在強勢族群中過濾：
- 股價突破近 20 日高點
- 當日量 > 5 日均量 × 1.5
- 5/10/20MA 多頭排列
- RSI(14) 在 50-75 之間
```

**Step 4：籌碼確認**
```
get_price_history + get_best_four_points 確認：
- 外資連續 3 日以上買超 → 優先
- 投信同向買超 → 加分
- 融資增加且股價上漲 → OK
- 大戶（400 張以上）持股比例上升 → 加分
```
