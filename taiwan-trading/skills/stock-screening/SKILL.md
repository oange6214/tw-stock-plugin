# 台股選股篩選工具

## 大盤環境分析

使用 tw-stock-agent 查詢：
- 加權指數與成交量趨勢
- 上漲/下跌家數比
- 外資期貨未平倉（多空判斷）
- VIX（恐慌指數）

### 市場環境分類
| 環境 | 判斷標準 | 操作建議 |
|------|---------|---------|
| 強勢多頭 | 指數站上 5/10/20MA，外資期貨多單 | 積極進場，部位可放大 |
| 盤整 | 指數在均線附近糾結 | 選強勢個股，部位縮小 |
| 偏弱 | 指數跌破 20MA，外資持續賣超 | 保守，減少新倉 |
| 空頭 | 指數大幅跌破均線 | 停止多單，等待落底 |

## 族群資金動能掃描

### 掃描流程
1. 查詢各族群近 1/5/20 日漲跌幅
2. 計算族群資金動能指數
3. 列出資金持續流入的前三大族群
4. 進入族群查看個股表現

### 重點關注族群（台灣市場）
- **科技：** 半導體、IC 設計、PCB、伺服器、記憶體
- **AI 相關：** CoWoS 封裝、HBM、AI 伺服器
- **電動車：** 電池材料、馬達控制、車用電子
- **綠能：** 太陽能、風電、儲能
- **傳產：** 航運、鋼鐵、石化（景氣循環）

## 個股技術篩選條件

使用 tw-stock-agent 查詢以下指標：

| 指標 | 篩選條件 | 查詢工具 |
|------|---------|---------|
| 股價突破 | 突破近 20 日高點 | `get_price_history` |
| 成交量 | 當日量 > 5 日均量 × 1.5 | `get_realtime_data` |
| 均線排列 | 5MA > 10MA > 20MA（多頭排列）| `get_best_four_points` |
| RSI(14) | 50–75 之間（強勢未過熱）| `get_best_four_points` |
| MACD | 黃金交叉或翻正 | `get_best_four_points` |
| 股價門檻 | > 10 元 | `get_stock_data` |
| 日均成交額 | > 1 億（近 3 個月均值）| `get_price_history` |
| 市值 | > 50 億 | `get_stock_data` |

### 篩選評分機制

| 條件 | 必要 / 加分 | 說明 |
|------|-----------|------|
| 突破 20 日高點 | 必要 ❌ 不符即排除 | 核心入場條件 |
| 量比 > 1.5x | 必要 ❌ 不符即排除 | 確認資金認同 |
| 均線多頭排列 | 必要 ❌ 不符即排除 | 趨勢確認 |
| RSI 50-75 | 必要 ❌ 不符即排除 | 強勢但未過熱 |
| MACD 黃金交叉 | +1 分 | 動能加分 |
| KD 低檔鈍化後向上 | +1 分 | 底部確認 |
| 布林通道突破上軌 | +1 分 | 突破力道強 |
| 外資連買 3 日以上 | +2 分 | 法人認同 |
| 投信同向買超 | +1 分 | 雙法人加持 |

**評分 ≥ 3 分** → 列入優先進場候選名單

## 輸出格式

掃描完成後輸出：
1. 大盤環境評估
2. 強勢族群排行（前 3 名）
3. 符合條件個股清單（附技術指標數值）
4. 籌碼面確認清單
