## 插件說明

### 1. taiwan-trading（主力插件）

台股兩週短波段交易系統，強調「技術突破 + 籌碼跟隨 + 嚴格風控」。

#### 指令
| 指令 | 說明 |
|------|------|
|  | 掃描全市場，選出 3-5 支符合短波段策略的標的 |
|  | 對單一標的進行完整波段分析 |
|  | 檢視當前持倉是否符合持有條件 |
|  | 執行風控檢查（停損/部位/曝險） |
|  | 分析大盤環境與主流族群 |

#### 技能（自動觸發）
- **short-wave-strategy**：完整兩週短波段策略邏輯（選股、進出場、風控）
- **stock-screening**：大盤環境分析、族群掃描
- **risk-management**：部位管理、停損停利執行規則

### 2. equity-research（個股研究）

| 指令 | 說明 |
|------|------|
|  | 執行選股篩選（配合兩週策略） |

|  | 季度財報分析報告 |
|  | 建立/更新投資論點 |
|  | 盤前晨報 |

### 3. financial-analysis（財務分析）

| 指令 | 說明 |
|------|------|

|  | 同業比較分析 |
|  | 競爭格局分析 |
|  | 財務模型稽核除錯 |

### 4. wealth-management（投資組合）

| 指令 | 說明 |
|------|------|
|  | 投資組合再平衡分析 |
|  | 財務規劃 |
|  | 投資績效報告 |
